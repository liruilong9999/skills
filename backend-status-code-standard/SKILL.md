# Backend Status Code Standard

## 🎯 适用场景（触发规则）

### ✅ 自动触发（必须应用）
当满足以下条件时自动启用本技能：

- 新编写后端接口代码
- 新建 API / Controller / Handler
- 从零设计接口返回结构
- 用户描述“写一个接口 / 实现接口 / 新增接口”

---

### ❌ 默认不触发（除非明确指定）

以下场景**默认不使用本技能**：

- 修改已有代码
- 重构已有接口（未明确要求统一返回结构）
- 修复 bug
- 局部逻辑调整

👉 **例外（强制触发）**：
如果用户明确说明：

- “按统一返回格式修改”
- “使用标准状态码”
- “套用 response 规范”

则必须启用本技能。

---

## 📐 返回结构（强制）

所有接口必须返回统一 JSON 结构：

```json
{
  "code": 200,
  "message": "OK",
  "data": {}
}
